# QUANTAXIS DEMO

quantaxis的一些使用示例


## 使用方法:

先将此项目 ```FORK```到你自己的账户中


```bash
git clone https://github.com/[你的github账号]/qademo
```

## 后台运行脚本

```
sudo chmod -R 777 ./qademo
cd qademo
sudo nohup ./startjupyter.sh &
```

## 更新代码

通过从 yutiansut/qademo 向你的项目推```pull request```来对项目进行更新

## 贡献代码

通过从 你的项目 向 yutiansut/qademo 推 ```pull request``` 来贡献你的代码
